Introduction

Modern E-commerce Platform is a complete solution for creating your online store. Built with Node.js, Express, and MySQL, it provides all the essential features needed for a successful e-commerce business.

Features

  • User Authentication System
    • Secure registration and login
    • Password recovery
    • Social media integration ready
  • Product Management
    • Category management
    • Product variants
    • Stock management
    • Image optimization
  • Shopping Cart
    • Real-time updates
    • Save for later
    • Guest cart
  • Order Management
    • Order tracking
    • Invoice generation
    • Email notifications
  • Payment Integration
    • Multiple payment gateways
    • Secure checkout
    • Payment status tracking

Installation

Prerequisites

  • Node.js (v14 or higher)
  • MySQL (v8 or higher)
  • npm (Node Package Manager)

Step-by-Step Installation

# 1. Extract the files
unzip modern-ecommerce.zip

# 2. Navigate to the project directory
cd modern-ecommerce

# 3. Install dependencies
npm install

# 4. Configure database
mysql -u root -p < database/schema.sql

# 5. Update configuration
cp config/db.example.js config/db.js
# Edit config/db.js with your database credentials

# 6. Start the server
npm start

Configuration

Database Configuration

// config/db.js
module.exports = {
    host: 'localhost',
    user: 'your_username',
    password: 'your_password',
    database: 'shopping_db'
}

Environment Variables

Variable Description Default
PORT Server port number 3000
NODE_ENV Environment (development/production) development
SESSION_SECRET Session encryption key your-secret-key

Customization

Themes

The platform comes with three pre-built themes:

  • Modern (default)
  • Classic
  • Minimal

To change the theme, modify the following in your HTML:

<link rel="stylesheet" href="/css/themes/[theme-name].css">

Custom Styling

Create your custom theme by copying and modifying the default theme:

cp public/css/themes/modern.css public/css/themes/custom.css

API Documentation

Authentication Endpoints

Endpoint Method Description
/api/auth/register POST Register new user
/api/auth/login POST User login
/api/auth/logout POST User logout

Product Endpoints

Endpoint Method Description
/api/products GET Get all products
/api/products/:id GET Get product by ID
/api/products/category/:category GET Get products by category

Troubleshooting

Common Issues

  1. Database Connection Errors 
    Error: ER_ACCESS_DENIED_ERROR
Solution: Check your database credentials in config/db.js
  2. Port Already in Use 
    Error: EADDRINUSE
Solution: Change the port in .env or terminate the process using that port
